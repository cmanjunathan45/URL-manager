Hello guys..!
This guide will helps you to how to use this URL manager

Recommended modules
----------- -------
pyfiglet
time
pyshorteners
terminal_banner
termcolor

to install the required modules run the command
pip install -r requirements.txt

This URL manager has 2 parts...
1)URL shorteners 
2)URL expander
.---------------.
| URL shorteners|
'---------------'
URL shorteners are used to shorten the long URL to short one...
 Here I code 16 URL shorteners
1.TINYURL
2.BIT.LY
3.CHILP.IT
4.CLCK.RU
5.CUTT.LY
6.GIT.IO(ONLY SHORTEN GITHUB URLS)
7.DA.GD
8.NULLPOINTER
9.OS.DB
10.OW.LY
11.PO.ST
12.ADFLY
13.QPS.RU
14.SHORT.CM
15.TINY.CC
16.IS.GD

.--------------.
| URL expanders|
'--------------'
URL Expander are used to expand the shortened URL to corresponding real longer URL one...
 Here I code 16 URL expanders
 
1.TINYURL
2.BIT.LY
3.CHILP.IT
4.CLCK.RU
5.CUTT.LY
6.GIT.IO(ONLY SHORTEN GITHUB URLS)
7.DA.GD
8.NULLPOINTER
9.OS.DB
10.OW.LY
11.PO.ST
12.ADFLY
13.QPS.RU
14.SHORT.CM
15.TINY.CC
16.IS.GD

Here we can expand the URL based on our choice 
For example 
Shortend URL:
https://po.st/test --> we need to expand this....
We should use po.st url exapnder only for expand otherwise it will not expand

API key

If any shorteners asks you for an API key ,go to the corresponding URL shorteners and make an account,then you will get your API key
